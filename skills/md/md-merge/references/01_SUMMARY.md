# Summary

The **md-merge** skill merges markdown chunks back into a single document.

## What It Does

- Reads numbered chunks in order
- Converts H1 back to H2 (reverses split promotion)
- Produces a single merged markdown file
- Validates chunk integrity before merge

## What It Is Not

- Not for arbitrary file concatenation
- Not for modifying content
- Not for restructuring documents
