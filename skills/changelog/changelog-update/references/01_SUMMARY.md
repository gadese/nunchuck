# Summary

Adds curated entries to the [Unreleased] section.

## When to Use

- User describes a change that should be documented
- PR is merged and needs changelog entry
- Bug fix or feature is complete

## Key Principle

**Curate, don't dump.** Transform commit messages into human-readable descriptions.
