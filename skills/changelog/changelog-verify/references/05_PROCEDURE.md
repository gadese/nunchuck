# Procedure

## Step 1: Run Verify

```bash
./skill.sh verify
```

Or specify path:
```bash
./skill.sh verify path/to/CHANGELOG.md
```

## Step 2: Review Output

If issues found:
- Review each reported issue
- Fix manually or use CLI commands
- Re-run verify to confirm

## Step 3: Report

Present findings to user with specific recommendations.
