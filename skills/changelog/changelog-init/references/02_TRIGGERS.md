# Triggers

## Invoke when

- Repository has no `CHANGELOG.md`.
- User asks to initialize or standardize a changelog.

## Do not invoke when

- User explicitly wants a different format than Keep a Changelog.

## Exit immediately if

- A changelog exists and the user does not confirm overwrite.
