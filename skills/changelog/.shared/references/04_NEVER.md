# Never

Forbidden behaviors for the changelog skillset.

## No Commit Dumps

- Never paste raw commit messages
- Never dump git log output
- Never auto-generate entries without curation

## No Format Violations

- Never use non-canonical categories
- Never use non-ISO date formats
- Never remove [Unreleased] section

## No Duplicate Spam

- Never add same entry twice
- Never add entry for already-referenced PR
- Never ignore duplicate warnings

## No Silent Changes

- Never edit changelog without user confirmation
- Never release without explicit version
- Never skip verification before release
