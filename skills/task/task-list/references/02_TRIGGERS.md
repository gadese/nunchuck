# Triggers

## Invoke when

- You need to see tasks and their derived warning flags.
- The user asks which tasks are open/closed/stale.

## Do not invoke when

- The user wants to create/select/close (use the corresponding skill).

## Exit immediately if

- `.tasks/` is missing and the user does not want to create tasks.
