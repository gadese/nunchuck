# Triggers

## Invoke when

- The user asks what the agent is currently allowed to assume
- The user asks to show the current disk-backed context contract
- The user wants a human-readable view of `.dtx/CONTRACT.yml`

## Do not invoke when

- The user is asking you to change the contract (use `dtx-forget` or other update-oriented skills)
