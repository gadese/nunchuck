# RPI Pipelines

Available pipelines for the `rpi` skillset.

## Available Pipelines

### 00_DEFAULT.md — Full R-P-I Workflow
Complete Research-Plan-Implement cycle with code review and commit message generation.
- Research → Plan → Implement → Code Review → Commit Message
- Includes mandatory pause points between phases
- Memory Bank integration at each phase

### 01_RESEARCH_ONLY.md — Research Phase Only
Execute only the research phase to document current codebase state.
- Produces research document in `llm_docs/research/`
- Useful for exploratory analysis before committing to implementation

### 02_PLAN_ONLY.md — Plan Phase Only
Execute only the planning phase based on existing research.
- Requires research document path as input
- Produces implementation plan in `llm_docs/plans/`

### 03_IMPLEMENT_ONLY.md — Implement Phase Only
Execute only the implementation phase based on existing plan.
- Requires plan document path as input
- Includes verification steps and progress tracking

### 04_FULL_WORKFLOW.md — Full R-P-I Workflow with Plan Review & Reimagine
Complete Research-Plan-Implement cycle with integrated plan review, plan reimagination, code review, and commit message generation.
- Research → Plan → Plan Review → Plan Reimagine → Implement → Code Review → Commit Message
- Includes mandatory pause points and context clearing between all phases
- Memory Bank integration at each phase
- Suitable for complex tasks requiring high quality and robustness
