# Always

## Must Preserve

- All decisions (explicit and implicit)
- All rationale (why decisions were made)
- All unresolved questions
- Named entities
- Conceptual ordering (sequence matters)
- Normative language

## Allowed Actions

- Remove exact duplicates
- Remove filler phrases
- Collapse repeated statements
- Consolidate near-identical phrasing
