# Taxonomy

Use these terms consistently in this skill:

## Plan

`docs/planning/phase-<N>/` with the root `plan.md`.

## Sub-plan

`docs/planning/phase-<N>/<letter>/` with a single `index.md` that lists all tasks with a brief description.

## Task

A roman numeral markdown file in a sub-plan (e.g., `i.md`, `ii.md`) representing an atomic unit of work.
