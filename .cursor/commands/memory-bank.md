# memory-bank

Manages workspace-specific Memory Bank files in `llm_docs/memory/`. Provides persistent cross-session context for AI agents. Global skill usable across multiple workspaces and projects.

This command delegates to the agent skill at `skills/memory-bank/`.

- **Path:** `skills/`

## Skill Location
