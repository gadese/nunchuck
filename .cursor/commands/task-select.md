# task-select

Select a task by writing its ID to `.tasks/.active`. Prints the selected task path and any suspicion flags (stale, hash mismatch) to encourage skepticism.

This command delegates to the agent skill at `skills/task/task-select/`.

## Skill Root

- **Path:** `skills/`

## Skill Location

- **Path:** `skills/task/task-select/`
- **Manifest:** `skills/task/task-select/SKILL.md`
