# task-list

List tasks in `.tasks/` with derived flags (stale, hash mismatch). Supports filtering by state and staleness. Stable ordering by updated_at.

This command delegates to the agent skill at `skills/task/task-list/`.

## Skill Root

- **Path:** `skills/`

## Skill Location

- **Path:** `skills/task/task-list/`
- **Manifest:** `skills/task/task-list/SKILL.md`
