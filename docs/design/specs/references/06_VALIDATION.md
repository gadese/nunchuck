# Validation Guidelines

- [ ] **Naming**: follows `<NN>_<NAME>.md` pattern
- [ ] **Frontmatter**:
  - [ ] Has required description field
  - [ ] Has required index field
- [ ] **Bodies**: Small and focused; less than 100 lines.
- [ ] Canonical file structure:
  - [ ] 00_ROUTER.md
  - [ ] 01_SUMMARY.md
  - [ ] 02_TRIGGERS.md
  - [ ] 03_ALWAYS.md
  - [ ] 04_NEVER.md
  - [ ] 05_PROCEDURE.md
  - [ ] 06_FAILURES.md
