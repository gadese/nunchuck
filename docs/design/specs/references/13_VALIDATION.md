# Validation Guidelines

- [ ] **Naming**: follows `<NN>_<NAME>.md` pattern
- [ ] **Frontmatter**: valid `YAML`, keys are from the allowed set
- [ ] **Bodies**: Small and focused; less than 100 lines.
- [ ] `uses[*]`: valid paths
- [ ] `executes[*]`: valid paths
- [ ] `produces[*]`: valid paths
- [ ] Canonical file structure:
  - [ ] 00_INDEX.md
  - [ ] 01_SUMMARY.md
  - [ ] 02_TRIGGERS.md
  - [ ] 03_ALWAYS.md
  - [ ] 04_NEVER.md
  - [ ] 05_PROCEDURE.md
  - [ ] 06_FAILURES.md
