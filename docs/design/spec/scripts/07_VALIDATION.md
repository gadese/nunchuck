# Validation Checklist

* [ ] CLI entrypoint exists for Unix and/or Windows
* [ ] `help`, `validate`, and `clean` (if applicable) are implemented
* [ ] `validate` is read-only
* [ ] No implicit dependency installation
* [ ] Hooks are optional and non-essential
* [ ] Dependency manifests live in `src/`
* [ ] Harness logic is separated from implementation logic
