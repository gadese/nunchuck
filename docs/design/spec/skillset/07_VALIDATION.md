# Validation

- [ ] `SKILL.md` frontmatter includes `metadata.skillset`.
- [ ] `metadata.skillset.schema_version` is present.
- [ ] `metadata.skillset.skills` contains only valid skill names.
- [ ] `SKILL.md` body contains a single H1 `INSTRUCTIONS`.
- [ ] `SKILL.md` body directs the reader to `pipelines/.INDEX.md`.
- [ ] `pipelines/` sibling directory exists.
- [ ] `pipelines/.INDEX.md` exists.
- [ ] `pipelines/00_DEFAULT.md` exists.
- [ ] All pipeline files follow the canonical pipeline file format.
- [ ] If present, `.shared/` contains only `scripts/`, `assets/`, and `references/`.
